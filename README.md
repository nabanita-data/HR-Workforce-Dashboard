# HR Workforce Dashboard

### Overview  
This Tableau project visualizes HR data generated synthetically in Python to simulate real-world workforce analytics. The dashboard presents employee distribution, salary comparisons, and hiring trends, offering a clear view of organizational structure and activity.

### Dashboard Features  
- **Employee Distribution by Department** – workforce breakdown by department  
- **Average Salary by Department** – compensation trends across teams  
- **Hiring Trend** – timeline of employee onboarding  
- **Summary Card** – total employees and average salary  

### Tools Used  
- **Python** (Faker, Pandas) for data generation and cleaning  
- **Tableau** for visualization and interactive dashboarding
